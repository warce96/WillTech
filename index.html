Propuesta de Desarrollo – Plataforma Médica Premium

Presentado a: Higort Fernández y José Rodríguez

Proveedor: Willtech

Duración estimada del proyecto: 4 meses

Resumen Ejecutivo

Willtech propone el desarrollo de una plataforma médica moderna y premium, compuesta por:

  • Aplicación móvil Android
  • Aplicación móvil iOS
  • Web App
  • Sitio público
  • Panel administrativo

Objetivo del Proyecto

Crear una herramienta profesional de consulta médica, con contenido exclusivo, suscripciones y accesos multiplataforma.

Alcance

  • Apps móviles completas (Android & iOS)
  • Web App con autenticación
  • Módulo de suscripciones
  • Buscador y categorización
  • Panel administrativo

Sistema de Suscripciones

Apps móviles

  • Google Play Billing
  • Apple In-App Purchases

Web (a definir)

  • Integración con Stripe, MercadoPago o Upay
  • Renovación automática
  • Webhooks para validación de pagos
  • Control de acceso premium

Tecnologías Propuestas

  • Flutter / React Native
  • React Web
  • Node.js o Python
  • PostgreSQL
  • AWS / Google Cloud

Condiciones Comerciales

Aporte inicial: USD 500

Este pago cubre:
  • Reserva de agenda
  • Inicio del diseño UX/UI (etapa donde se define la experiencia e interfaz del usuario)
  • Configuración técnica inicial

Resto de pagos

  • El saldo se pagará por hitos
  • No habrá costos adicionales sin aprobación

Cronograma – 4 Meses

  • Mes 1: UX/UI + Arquitectura
    • Etapa 1 — Día 15: Diseño UX/UI inicial
    • Etapa 2 — Día 30: Arquitectura técnica
  • Mes 2: Apps + Backend
    • Etapa 3 — Día 15: Base de apps móviles
    • Etapa 4 — Día 30: Desarrollo API y backend
  • Mes 3: Web + Pagos + Panel admin
    • Etapa 5 — Día 15: Web App + login
    • Etapa 6 — Día 30: Pagos y panel admin
  • Mes 4: QA + Publicación
    • Etapa 7 — Día 15: QA y optimización
    • Etapa 8 — Día 30: Publicación + capacitación

*El cronograma puede variar según solicitudes adicionales. Cualquier cambio se cotizará como FIX independiente.*

Funcionalidades No Incluidas

Estas funcionalidades requieren cotización adicional:

  • Videollamadas nativas
  • Pagos recurrentes avanzados
  • Integración con aseguradoras
  • Módulo contable / legal
  • Mantenimiento anual

Precio sujeto a variación según funcionalidades adicionales solicitadas por el cliente.
